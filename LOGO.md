# Irancell Anniversary Logo Elements
# This file describes the visual elements that would be used in the game

## Logo Concepts
- Yellow Pacman with Irancell branding
- Network connection symbols integrated into the maze
- Anniversary badges and decorations

## Color Scheme
- Primary: #FFCC00 (Irancell Yellow)
- Secondary: #0000FF (Blue for walls and UI elements)
- Accent: #FFD700 (Gold for anniversary elements)
- Background: #000000 (Black)

## Visual Elements
- Pacman character in Irancell yellow
- Ghosts with network-themed names
- Maze walls designed to look like network towers or connections
- Coins that look like small network signals
- Power pellets designed as network boosters

## Anniversary Themes
- "20 Years of Connection" messaging
- Network signal motifs throughout the game
- Celebration elements like confetti in the game colors
- Special anniversary achievements and badges

## Possible Sprite Descriptions
1. Player (Pacman): Yellow circle with animated mouth, Irancell logo on the back
2. Enemies (Ghosts): Classic ghost shape but with network signal icons on their fronts
3. Coins: Small yellow circles with antenna-like elements
4. Power Pellets: Larger yellow circles with signal wave patterns
5. Walls: Blue blocks with network tower silhouettes
